# 🎛️ LaserCalc Pro 管理后台

## 快速开始

### 1️⃣ 创建管理员账号

```bash
npm run create-admin
```

### 2️⃣ 启动开发服务器

```bash
npm run dev
```

### 3️⃣ 登录管理后台

访问: http://localhost:3000/admin/login

默认账号:
- 用户名: `admin`
- 密码: `admin123`
